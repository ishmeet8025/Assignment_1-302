## Project Overview

This is an **Android application** that displays a list of courses using **RecyclerView**.  
When a user taps a course, the app opens a detail screen showing the course code, name, and description.  
The app is built with **Kotlin** and follows **Material Design** principles for a modern, interactive look.  

---

## Features

- Display a list of **6 courses** with code and name.
- Clickable course items that navigate to **DetailActivity**.
- Detailed screen shows **course code, name, and description**.
- Clean **Material3 UI** with CardView, padding, and shadows.
- Fully documented code with comments for classes and methods.
